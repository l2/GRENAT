% MONO_NEXT_GREVLEX: grevlex next monomial.

  Discussion:

    For example:

    M = 3

    #  X(1)  X(2)  X(3)  Degree
      +------------------------
    1 |  0     0     0        0
      |
    2 |  0     0     1        1
    3 |  0     1     0        1
    4 |  1     0     0        1
      |
    5 |  0     0     2        2
    6 |  0     1     1        2
    7 |  1     0     1        2
    8 |  0     2     0        2
    9 |  1     1     0        2
   10 |  2     0     0        2
      |
   11 |  0     0     3        3
   12 |  0     1     2        3
   13 |  1     0     2        3
   14 |  0     2     1        3
   15 |  1     1     1        3
   16 |  2     0     1        3
   17 |  0     3     0        3
   18 |  1     2     0        3
   19 |  2     1     0        3
   20 |  3     0     0        3

    Thanks to Stefan Klus for pointing out a discrepancy in a previous
    version of this code, 05 February 2015.

    05 February 2015

  Author:

    John Burkardt

  Parameters:

    Input, integer M, the spatial dimension.

    Input, integer X(M), the current monomial.
    The first element in the sequence is X = [ 0, 0, ..., 0, 0 ].

    Output, integer X(M), the next monomial.

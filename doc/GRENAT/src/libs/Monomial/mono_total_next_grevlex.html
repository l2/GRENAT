DESCRIPTION 

*****************************************************************************80

% MONO_TOTAL_NEXT_GREVLEX: grevlex next monomial with total degree equal to N.

  Discussion:

    We consider all monomials in an M dimensional space, with total degree N.

    For example:

    M = 3
    N = 3

    #  X(1)  X(2)  X(3)  Degree
      +------------------------
    1 |  0     0     3        3
    2 |  0     1     2        3
    3 |  1     0     2        3
    4 |  0     2     1        3
    5 |  1     1     1        3
    6 |  2     0     1        3
    7 |  0     3     0        3
    8 |  1     2     0        3
    9 |  2     1     0        3
   10 |  3     0     0        3

  Licensing:

    This code is distributed under the GNU LGPL license.

  Modified:

    09 September 2014

  Author:

    John Burkardt

  Parameters:

    Input, integer M, the spatial dimension.

    Input, integer N, the degree.
    0 <= N.

    Input, integer X(M), the current monomial.
    To start the sequence, set X = [ 0, 0, ..., 0, N ].

    Output, integer X(M), the next monomial.
    The last value in the sequence is X = [ N, 0, ..., 0, 0 ].

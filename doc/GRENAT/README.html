0001 GRENAT 
0002 =======
0003 GRENAT  = **GR**adient **EN**hanced **A**pproximation **T**oolbox
0004 
0005 
0006 GRENAT regroups many techniques for generating surrogate models. These surrogate models can be built using responses and gradients obtained at some sample points.
0007 
0008 The available surrogate models are
0009 
0010 * Radial Basis Function (RBF/GRBF)
0011 * Kriging/Cokriging (KRG/GKRG)
0012 * Support Vector Regression (SVR/GSVR)
0013 
0014 This toolbox is the result of the PhD thesis of [Luc LAURENT](http://www.lmssc.cnam.fr/fr/equipe/luc-laurent) ([LMSSC](http://www.lmssc.cnam.fr)), supervised by [Pierre-Alain Boucard](http://w3.lmt.ens-cachan.fr/site/php_perso/perso_page_lmt.php?nom=BOUCARD) ([LMT-Cachan](http://www.lmt.ens-cachan.fr/)) and [Bruno Soulier](http://w3.lmt.ens-cachan.fr/site/php_perso/perso_page_lmt.php?nom=SOULIER) ([LMT-Cachan](http://www.lmt.ens-cachan.fr/)). 
0015 
0016 Features
0017 ------
0018 GRENAT is able to 
0019 
0020 * Normalize the input data
0021 * Build the surrogate models
0022 * Estimate hyperparameters (using optimizer and dedicated criteria)
0023 * Evaluate the surrogate model at sample and non-sample points (and calculate the gradients of the approximation)
0024 * Compute the variance of the prediction
0025 * Compute the Cross-Validation of the surrogate model
0026 * Compute the Expected Improvement (and derivated formulations)
0027 
0028 External included toolboxes
0029 ------
0030 
0031 GRENAT uses the
0032  
0033 * [MultiDOE](https://bitbucket.org/luclaurent/multidoe) toolbox
0034 * A modified version of the Particle Swarm Optimization Toolbox [PSOt](http://www.mathworks.com/matlabcentral/fileexchange/7506-particle-swarm-optimization-toolbox)
0035 
0036 These toolbox are included in GRENAT.

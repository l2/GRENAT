0001 GRENAT 
0002 =======
0003 GRENAT  = **GR**adient **EN**hanced **A**pproximation **T**oolbox
0004 
0005 
0006 GRENAT regroups many techniques for generating surrogate models. These surrogate models can be built using responses and gradients obtained on some sample points.
0007 
0008 The available surrogate models are
0009 
0010 * Radial Basis Function (RBF/GRBF)
0011 * Kriging/Cokriging (KRG/GKRG)
0012 * Support Vector Regression (SVR/GSVR)
0013 
0014 
0015 Features
0016 ------
0017 GRENAT are able to 
0018 
0019 * Normalize the input data
0020 * Build the surrogate models
0021 * Estimate hyperparameters (using optimizer and dedicated criteria)
0022 * Evaluate the surrogate model at sample and non-sample points (and calculate the gradients of the approximation)
0023 * Compute the variance of the prediction
0024 * Compute the Cross-Validation of the surrogate model
0025 * Compute the Expected Improvement (and derivate formulations)
0026 
0027 External included toolboxes
0028 ------
0029 
0030 GRENAT uses the
0031  
0032 * [MultiDOE](https://bitbucket.org/luclaurent/multidoe) toolbox
0033 * A modified version of the Particle Swarm Optimization Toolbox [PSOt](http://www.mathworks.com/matlabcentral/fileexchange/7506-particle-swarm-optimization-toolbox)
0034 
0035 These toolbox are included in GRENAT.
